# vylo A work in progress Discord bot created in Python using discord.py library. 
<br> 
## Current Features: 1. Member Greeting - Automatically greets users when they join the server. - Sends a friendly welcome message. 2. Role Assignment - Assigns roles to members with a simple command. - Useful for moderation.
